# Finre Teslimat Asamalari

Bu plan, urunun hangi adimda ne gosterecegini netlestirir.

## Asama 1 - Cekirdek motor (tamam)

- Soru bankasi var
- Skor motoru var
- Kural tabanli yonlendirme var

## Asama 2 - API (tamam)

- Questions ucu var
- Assessment ucu var
- Session acma ve cevap kaydetme uclari var

## Asama 3 - Wizard arayuzu (tamam)

- Adim adim soru akisi var
- Sonuc ekrani var
- API yoksa fallback hesap var

## Asama 4 - Gorsel paylasim noktasi

Bu asamada site gorseli paylasilir.
Neden: artik akisin basi ve sonu calisir, ekranda gercek veri gorulur.

Paylasilacaklar:
- Ana wizard ekrani
- Sonuc ekrani
- Mobil gorunum

## Asama 5 - Son polish

- KVKK ve acik riza metni
- Metin duzeltmeleri
- Son UI ince ayar

Bu asama bitince final gorsel tekrar paylasilir.
